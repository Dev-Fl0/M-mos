type de sélecteurs

ID:

Dans un premier temps dans la balise précédent l'endroit ou l'on veux mettre l'ID: 
          <h3 id="Nom de l'ID">titre</h3>
Ensuite lien vers cet ID: 
          <a href="#Nom de l'ID" title="Aller à Nom de l'ID(popup au survol du lien)">Nom de l'ID</a>

Classes:

Dans un premier temps dans la balise précédent l'endroit ou l'on veux mettre la classe: 
          <h3 class="Nom de la classe">titre</h3>
Ensuite lien vers cette classe: 
          <a href=".Nom de la classe" title="Aller à Nom de la classe(popup au survol du lien)">Nom de la classe</a>

Récapitulatif des sélecteurs:

La sélection par la balise:
p {}

La multi-sélection:
h1, em { }

La sélection par la classe:
.class { }

La sélection par l'ID:
#id { }

Le sélecteur universel (toutes les balises):
* { }

La balise B contenue dans A:
A B { }

La balise a qui a un attribut title:
a[title] { }

La balise a qui a un attribut title qui a un valeur "Accueil":
a[title="Accueil"] { }

La balise a qui a un attribut title dont le mot "Accueil" est en tout ou partie la valeur:
a[title*="Accueil"] { }

Lignes de commandes:

Taille des polices:

Par défaut les texts sont à 16px.
Taille absolue(ne prend pas en considération l'utilisateur et c'est en pixel):

selécteur { font-size:1px ;}

Taille relative:

sélecteur { font-size: xx-small/x-small/small/medium/large/x-large/xx-large }

Avec em(1 em = normal):

sélecteur { font size: 1em ;}

En pourcentage:

sélecteur { font-size: 1ex;}

Changer la police:

En téléchargant une police:
Sur dafont(à mettre dans le dossier src et déclarer sur CSS):

@font-face { font-family: 'nom de la police'; src: url('../src/nom de la police') format('truetype';)}

sélecteur { font-family: nom de la police, autre police , sans-serif; }

Depuis un url:
Depuis google fonts(séléctioner les polices et copier l'url et déclarer sur la page html dans le head):

sélecteur { font-family: 'nom de la police' , autre police , sans-serif ; }

FORME:

En majuscules:

sélecteur { text-transform: uppercase; }

En gras:

sélecteur { font-weight: bold; }

En italique:

sélecteur { font-style: italic; }

Souligner:

sélecteur { text-decoration: underline; }

Enlever la décoration des liens:

sélecteur(a) { text-decoration: none; }

MISE EN PAGE:

Alignement:

sélecteur { text-align: center, right, justify; }

Aligner une liste:
Vérifier le padding sur la console de google(inspecter):

sélecteur { padding: 0; }

Enlever les point des listes
Voir plusieurs styles ici:

sélecteur(li) { list-style-type: none; }

Mettre des chiffres:
li { list-style-type: decimal; }

Mettre des chiffres romains:
li { list-style-type: upper-roman ou lower-roman; }

Mettre des cercles vide:
li { list-style-type: circle; }

LES FLOTANTS:

Rendre l'image ignorée par le text:

sélecteur(logo) { float: left, right; }

Stoper un élément flotant:
Il doit avoir la même valeur que le flotant:

sélecteur { clear: left, right, both; }

Modifier la couleur des éléments et fond d'écran:

Mettre en couleur le fond de la page:
Couleurs par défaut:

sélecteur(body) { background-color: couleur; }

Avec une transparence:

sélecteur(body) { background-color: rgba(0, 0, 0, 0); }

Mettre un fond d'ecran

body { background-img: url('lien de l'image'); }

Réglages du fond d'écran:

Le fond d'écran bouge en descendant:
body { background-attachment: scroll; }

Le fond d'écran ne bouge pas c'est le texte qui bouge:
body { background-attachment: fixed; }

Répetition du fond d'écran:

Le fond d'écran est répété:
body { background-repeat: repeat; }

Le fond d'écran n'est pas répété:
body { background-repeat: no-repeat; }

Le fond d'écran est répété sur l'axe y ou x:
body { background-repeat: repeat-y, repeat-x; }

Position du fond d'écran:

Positionnement en pixel:
body { background-position: repeat; 100px 100px; }

Positionnement en adjectifs:
body { background-position: bottom/top/center, bottom left/rigth/center, }

En une seule propriété:

body { background: url('lien du background') no repeat center center; }

Plusieurs fond d'écrans:

body { background: url('lien du background') no repeat center center, url('lien du background') fixed no repeat center center; }
